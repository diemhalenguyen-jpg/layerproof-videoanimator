Reveal Builder

Workflow guide

A step-by-step walkthrough — from importing your slides to running a live synced presentation.

Photos → Audio → Transcript → Draw regions → Map timing → Sticker → Caption → Present
1

Import

Load your slide images

On the home screen, click the drop zone or drag your images directly onto it. Images are displayed in 9:16 format — any aspect ratio is supported and fitted inside the frame.

Files are sorted by name automatically, so naming them 1.png, 2.png, 3.png keeps them in order.

Tip If you need to add more images later, use the + Add images button at the bottom of the filmstrip. You can also drag thumbnails to reorder slides, or click the × to delete one.
2

Import

Upload your voiceover audio

In the editor, click ♪ Audio in the top bar and select your audio file. The player will appear in the presentation bar once loaded.

Any common audio format works — MP3, M4A, WAV, etc.

Why audio first? The audio file is needed before transcription. It's also used to drive auto-reveal timing during the presentation — regions appear in sync with the voiceover.
3

Transcribe

Generate a timed transcript

Enter your Gemini API key in the top-right input, then click ⚡ Transcribe. The audio is sent to Gemini, which returns a list of short timestamped segments — usually in a few seconds.

  • Each segment appears in the Transcript panel on the right
  • Click any segment text to edit it — useful for fixing spelling or adjusting captions
  • Click a segment row to assign it as the trigger time for the currently selected region
Note Your Gemini key is saved locally in the browser — you won't need to paste it again next time.
4

Draw regions

Draw reveal rectangles on the slide

Click and drag anywhere on the slide canvas to mark a region. Each rectangle defines one element that will be progressively revealed during the presentation.

You can draw as many regions as you need per slide — they will be revealed in the order you drew them. The numbered badge in the top-left of each region shows the reveal order.

  • Regions are listed in the Reveal order panel on the right
  • Click a region in the list to select it and adjust its settings
  • Use Reset crops in the top bar to clear all regions on the current slide
Tip Use the Exposure and Blacks sliders in the Image panel to adjust contrast — great for darkening grey areas on slides with dark backgrounds.
5

Map timing

Set when each region appears

Select a region from the Reveal order list, then click a row in the Transcript panel to assign that timestamp as the trigger. The region will appear at that exact moment during playback.

You can also type a time directly in the trigger at field using M:SS format.

  • Triggers must increase — a region can't appear before the previous one
  • The keyword field is optional — it stores a phrase hint for script-mode matching
  • In Script mode, timing is assigned automatically when you draw each region, based on phrase matching in the transcript
Script mode If your script contains tags like [S2: phrase to find], switch to Script mode on the home screen before loading images. Regions will be auto-matched to the phrase in the transcript as you draw them — no manual timing needed.
6

Sticker

Choose a cat sticker

Click 📁 Load in the Cats panel to pick a folder of PNG or GIF images. The sticker will animate in each time a new region is revealed during the presentation.

Select a region, then click a cat thumbnail to assign it to that region. You can assign a different sticker to each reveal, or check Same cat to inherit the one from the previous region.

  • Drag the cat on the canvas to reposition it
  • Use the size % input next to the Load button to scale all stickers
  • The cat folder is remembered — it auto-loads next time you open the tool
7

Caption

Set up live captions

Captions are on by default and shown during presentation — they display the current transcript segment in sync with the audio. No extra setup is needed beyond having a transcript.

  • Adjust Font size to fit your content
  • Use the Position slider to move the caption up or down in the frame
  • Toggle captions off using the Show captions checkbox or the CC button in the presentation bar
  • Edit any segment text in the Transcript panel to fix wording before presenting
Tip Captions run independently of the slide reveals — they keep updating even after all regions on a slide have appeared.
8

Present

Run the presentation

Click Present → in the top bar. The canvas fills the screen. Press play on the audio and regions will reveal automatically in sync with the voiceover.

You can also advance manually at any time:

  • Space — reveal the next region
  • ← — go back one step
  • Esc — return to the editor
  • Click anywhere on the slide to advance
Save your work Use 💾 Save to export the project as a .json file — it includes all regions, timing, transcript, and the images. Open it again with 📂 Open to continue where you left off.
